Descrição:
Banco de dados com registros da missão. Acesso protegido por senha.

Comandos possíveis:
- usar senha / aplicar senha / acessar com senha
- ler registros / consultar registros / acessar registros
- baixar arquivos / coletar arquivos / copiar arquivos

Comandos compostos:
- usar senha do diário para acessar registros
- baixar arquivos sobre criatura

Eventos aleatórios:
- Interferência magnética (-10 saúde)
- Alerta de intrusão (-5 saúde)

Sistema de saúde:
- Saúde inicial: 100

Itens coletáveis:
- Registros digitais

Diálogos ramificados:
- Terminal: “Acesso concedido. Dados confidenciais disponíveis.”
