Descrição:
Ambiente virtual usado para treinar comunicação com espécies alienígenas.

Comandos possíveis:
- iniciar simulação / ativar simulação / ligar simulação
- falar com criatura / conversar com criatura / comunicar com criatura
- examinar ambiente / observar ambiente / ver ambiente

Comandos compostos:
- iniciar simulação com linguagem simbólica
- falar com criatura sobre ciclo

Eventos aleatórios:
- Sobrecarga neural (-20 saúde)
- Falha de renderização (-10 saúde)

Sistema de saúde:
- Saúde inicial: 100

Itens coletáveis:
- Linguagem simbólica

Diálogos ramificados:
- Criatura: “Vocês quebraram o ciclo. Reconstruir ou apagar?”
